# Давлат синови илмий ҳисобот генератори

Ўсимликларни ҳимоя қилиш воситалари **давлат синови илмий ҳисоботини** (.docx)
автоматик яратувчи веб-платформа. Фойдаланувчи дала ўлчов маълумотларини киритади —
тизим биологик самарадорлик, статистик таҳлил ва графикларни ҳисоблаб, расмий
тузилишдаги Word ҳужжатини тайёрлайди.

> **Ўзбекистон Республикаси · Ўсимликлар карантини ва ҳимояси агентлиги** учун мўлжалланган.

---

## ⚠️ Маълумотлар ишончлилиги (муҳим)

Тизим **фақат фойдаланувчи киритган дала ўлчовларидан** ҳисоб-китоб қилади:

- Биологик самарадорлик, статистика (ANOVA, НСР₀.₀₅) ва графиклар — киритилган
  зараркунанда сони / касаллик индекси / бегона ўт зичлиги ва ҳосилдорлик
  маълумотларидан **ҳисобланади**.
- Тизим дала ўлчовларини (сонлар, ҳосилдорлик) **ўзи ўйлаб чиқармайди**.
- Матн қолиплари киритилган препарат/экин/зарарли организмга мослаштирилади,
  аммо натижа рақамлари фақат ҳақиқий киритилган маълумотдан келиб чиқади.

Бу — ҳисоблаш ва расмийлаштириш воситаси, синов маълумотларини сохталаштириш
воситаси эмас.

---

## Асосий имкониятлар

- **Препарат турини автоматик аниқлаш** — таъсир этувчи модда номидан (лотин ёки
  кирилл ёзувда): гербицид, инсектицид, фунгицид, акарицид, нематицид, биопрепарат,
  родентицид, дефолиант, десикант.
- **Биологик самарадорлик** — препарат турига қараб мос методика:
  - Инсектицид/акарицид/биопрепарат — **Henderson–Tilton (1955)** ёки **Abbott (1925)**.
  - Фунгицид — касаллик ривожланиш индекси бўйича (EPPO / давлат методикаси).
  - Гербицид — бегона ўтлар зичлиги бўйича (давлат гербицид синов методикаси).
- **Статистик таҳлил** — тасодифий блокли дала тажрибаси ANOVA (Доспехов):
  НСР₀.₀₅, Sx, Sd, CV%, F-мезон, P-қиймат.
- **Графиклар** — самарадорлик, зарарли организм динамикаси, ҳосилдорлик (PNG,
  ҳужжатга киритилади).
- **.docx ҳисобот** — 16 бўлимли расмий тузилиш (титул, тасдиқ, мундарижа, кириш,
  адабиётлар шарҳи, синов баёномаси, шароит, методика, натижалар, самарадорлик
  жадваллари, ҳосилдорлик, муҳокама, хулоса, тавсия, давлат рўйхати хулосаси,
  далолатнома).
- **Шаблон таҳлили** — намуна .docx юкланса, унинг тузилиши таҳлил қилинади.

## Технологиялар

- **Next.js 16** (App Router) + **React 19** + **TypeScript**
- **Tailwind CSS v4**
- `.docx` генерация — **docx** (+ `docxtemplater`/`pizzip` шаблон таҳлили учун)
- Графиклар — SVG → PNG (**sharp**)
- Статистика — **jstat** (t/F тақсимот квантиллари)
- Тестлар — **vitest**

## Локал ишга тушириш

Талаблар: Node.js 20+ (лойиҳа Node 22'да синовдан ўтган).

```bash
cd trial-report-generator
npm install
npm run dev        # http://localhost:3000
```

Ишлаб чиқариш режими:

```bash
npm run build
npm start
```

Тестлар:

```bash
npx vitest run
```

## API

Барча йўллар `POST` (Node.js runtime):

| Йўл | Вазифа |
|-----|--------|
| `/api/detect` | Таъсир этувчи моддадан препарат турини аниқлаш |
| `/api/compute` | Дала маълумотларидан самарадорлик + статистикани ҳисоблаш (JSON) |
| `/api/report` | Тўлиқ .docx ҳисоботни яратиш ва қайтариш |
| `/api/template` | Юкланган намуна .docx тузилишини таҳлил қилиш |


## Методика манбалари

- Abbott W.S. (1925). A method of computing the effectiveness of an insecticide.
- Henderson C.F., Tilton E.W. (1955). Tests with acaricides against the brown wheat mite.
- Доспехов Б.А. Методика полевого опыта.
- EPPO PP1 стандартлари (фунгицид самарадорлигини баҳолаш).
- Ўзбекистон Республикаси давлат синов методикалари.
